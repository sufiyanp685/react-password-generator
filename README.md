# 🔐 React Password Generator

A secure and customizable password generator built using React. It allows users to generate strong passwords with different character options and copy them instantly.

---

## 🚀 Features

- 🔢 Adjustable password length
- 🔤 Include uppercase & lowercase letters
- 🔣 Include numbers and special characters
- ⚡ Instant password generation
- 📋 Copy to clipboard functionality
- 🎯 Simple and clean UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Tailwind CSS 

---



## 💡 What I Learned

- Managing state using React Hooks
- Handling user input dynamically
- Generating secure random data
- Clipboard API usage in JavaScript
